My first game in Unity

Posted by Devesh Paragiri on August 24, 2018

My Dream come true

As a child I had always loved playing games.I used to really wonder how it was possible to create a game which was so immersive and fun to play.When I came to know about the game development workshop at IITM Research Park I was really excited and signed up for the workshop immediately.The workshop was from 9:00 AM to 6:00 PM and we started with the basics of Unity.I was shocked to know about the number of hours spent to create a single game.The amount of hardwork to create a simple game such as Play Snake.

The mentors at the workshop were very kind and clarified my doubts.The most challenging part was aligning the sprites into place as it needed exact X and Y coordinates.The coding was done in C Sharp.I felt very happy when the ball moved from point A to B.I liked Unity because it is easy to debug,use and design.I felt very happy when I finished making the game and look forward to do more!

Rules of the Game

The rules of the game are very simple.The user has to jump across chainsaws and reach the end.If the ball touches the chainsaw he loses the game and must play again from the beginning.
