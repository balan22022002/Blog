#Codethon

Posted by Devesh Paragiri on August 24, 2018

I was all excited for the Codethon which was hosted by Thinkdiff S'kool.I was shocked at the number of kids who had turned up on a holiday.I like codethons the most because you have the freedom to chose what you want to do and can code in any programming language.I chose to do a simple game with javaScript.I faced a lot of challenges as I was relatively new to JavaScript and had a hard time coping up with the syntax.I used p5 and a simple html page to host my game.

The game has been hosted in my website deveshparagiri.com.My mentor Ashish Cherian helped me with the coding part.
